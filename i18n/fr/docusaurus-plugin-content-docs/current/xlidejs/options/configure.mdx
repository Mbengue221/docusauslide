title: Spécifier une option
sidebar_label: xlidejs - Les options (suite)

---

Pour ajouter une option a votre slide vous utiliserez la methode `addOption` de votre objet `xlide`. 
Elle prend comme arguments `option` (le nom de l'option) et `value` la valeur à assigner: 
```js
slide.addOption(option,value)
```

**Par exemple si je veux changer la duree entre deux transitions:**
```js 
slide.addOption('interval',3)
```
***Notez que `value` a pour valeur par defaut true*, qu'importe l'option specifie, voyez le code ci-dessous**
```js
slide.addOption('previews',true)
```
Il peut etre ecrit ainsi
```js
slide.addOption('previews')
```
          
Je peux aussi spécifier plusieurs options en meme temps
